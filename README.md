## Escenario elegido
Escenario B – Análisis de Ventas de una Pequeña Empresa

## Descripción del dataset
Se utilizó un dataset simulado de ventas de productos deportivos (camisetas de fútbol).
Contiene registros con producto, cantidad vendida, precio unitario y fecha de venta.
El archivo se encuentra en la carpeta /datos.

## ¿Qué hace el script?
El script analisis_datos.py procesa el dataset y calcula:
- Ventas totales del período
- Producto más vendido
- Ventas por mes
- Genera un gráfico de evolución de ventas guardado en /resultados

## Cómo ejecutarlo
1. Abrir Google Colab
2. Clonar el repositorio con git clone
3. Instalar dependencias: pip install pandas matplotlib
4. Ejecutar: python scripts/analisis_datos.py
